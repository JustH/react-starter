UW Foster Web App Style Guide

The purpose of this HTML is to help determine what default settings are in the Foster Web App Style guide.

This web page is an overview of the Fonts and Colors used for the Foster School of Business Site design. The Fonts and Colors chosen comply with the existing University of Washington Brand Guidelines and are intended to guide proper design and programming of the website to ensure a clear and consistent appearance to users

It is built with Bitters, Bourbon, and Neat. All possible HTML Elements are included in this HTML so as to not miss any possible Elements when designing a site.
